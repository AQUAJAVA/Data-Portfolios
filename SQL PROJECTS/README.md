## 1. STUDENT PERFORMANCE ANALYSIS :-
This project is a basic SQL-driven analysis of student performance data focused on understanding patterns through segmentation rather than forced correlations. Using MS SQL Server, the analysis explores how study hours, attendance, grades, and score distributions relate to overall performance. Since the dataset is highly synthetic and evenly balanced, demographic variables offered little insight, so the focus shifted to grade-wise, distribution-based, and percentile-based patterns. Although the project is introductory in nature, it provided a strong lesson in data quality assessment and the importance of adapting analytical approaches to the nature of the data rather than the analystâ€™s expectations.

## 2. SQL CHALLENGE (PRACTICE PROJECT) :-
This project documents my hands-on practice in data analytics and SQL using real-world datasets. I focused on cleaning messy data with Python (Pandas), understanding how Python connects to Microsoft SQL Server, and solving structured SQL questions with increasing complexity. Instead of following tutorials or synthetic problems, I worked with public datasets that required real data-cleaning and logical thinking. The goal was to strengthen my fundamentals, improve analytical reasoning, and build confidence working with realistic data scenarios.
