## Project 1: Madhav E-Commerce Sales Dashboard

This is my **first ever Power BI dashboard**, created by following a YouTube tutorial and replicating the analysis end-to-end to understand how real-world dashboards are built.

**Tutorial Reference:**  
[Madhav E-Commerce Sales Analysis – Power BI](https://www.youtube.com/watch?v=6cV3OwFrOkk)

---

## Dataset Description

The analysis is based on two tables:

###  Orders Table  
Contains order-level information:
- Order ID  
- Order Date  
- Customer Name  
- State  
- City  

### Details Table  
Contains transactional details:
- Order ID  
- Amount  
- Profit  
- Quantity  
- Category  
- Sub-Category  
- Payment Mode  

These tables are connected using **Order ID**.

---

## Dashboard Highlights

The dashboard provides a complete overview of sales and profit performance with the following visuals:

- **KPIs**
  - Total Orders  
  - Total Sales  
  - Total Profit  

- **Sales Analysis**
  - Sum of Amount by Category  
  - Sum of Amount by Sub-Category  

- **Geographical Insights**
  - Top 5 States by Sales  

- **Customer Insights**
  - Top 5 Customers by Sales  

- **Payment Analysis**
  - Sales distribution by Payment Mode  

- **Time-Based Analysis**
  - Monthly Profit Trend  

- **Interactivity**
  - Quarter-wise slicer (Q1–Q4)
